GitHub Community SRM

GitHub Community SRM, the dynamic and student-led community affiliated with GitHub at SRMIST KTR. We're all about leading the way in the open source world. They're on a mission to revolutionize the way our college engages with open source technology. They're here to spread the word about the power of GIT and GitHub, cultivating a vibrant ecosystem of collaboration and innovation. They organize events, workshops and sessions and also equip you with skills needed to conquer the open source realm.

Team Domains are:
  • Technical
  • Event Management
  • Sponsorships
  • Design
  • Editorial
  • Community Outreach
